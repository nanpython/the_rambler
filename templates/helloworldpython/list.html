List

A list is a collection of elements of any type.

Note: Use the embedded python interpreter at the bottom of this page to try the examples.

Examples

Create a list

Create a list of names. 
names = ['modi', 'saboo', 'kumar', 'sam', 'ali']
 Create an empty list of entries. 
entries = []
 The elements in a list are separated by commas. Elements can be a string, a number, a boolean value or any object including None.

Print the list to the console using the print function. 

print(names)
Output 
['modi', 'saboo', 'kumar', 'sam', 'ali']
 A list may alsobe created from the list function. 
name = 'anand'
letters = list(name)
print(letters)
Output 
['a', 'n', 'a', 'n', 'd']

Add an element

Elements can be added or removed from a list. Use the append function to add items to a list.

names.append('hari')
 The element is added at the end of the list. 
['modi', 'saboo', 'kumar', 'sam', 'ali', 'hari']

Remove an element

To remove kumar from the list, use remove function. It removes the first occurence in the list. 
names.remove('kumar')

Modify a list

Modify a list. To replace an element in a list with another item, we need to use the index operator. A list arranges its elements in a sequence starting from 0. So, the first element is at ineex 0 and the index increases until the last element is at index length-of-the-list minus 1. 
len(names)
 Output 
5
 Replace saboo with pavan 
names[1] = 'pavan'
 Print the names again. The output is 
['modi', 'pavan', 'sam', 'ali', 'hari']
 To remove all elements from a list, use the function clear or simply assign an empty list or another list of elements to replace all. 
names_2 = names
 Now names_2 contains all the elements of the list names.

Merge lists

To merge a list with another list, use the extend function.

new_names = ['sarah', 'rama', 'agha']
names.extend(new_names)
print(names)
Output 
['modi', 'pavan', 'sam', 'ali', 'hari', 'sarah', 'rama', 'agha']
 To add an element at a particular location in a list, use the insert function. After insertion, the elements will be pushed to the right after the specified index. 
names.insert(3, 'raja')
print(names)
 Output 
['modi', 'pavan', 'sam', 'raja', 'ali', 'hari', 'sarah', 'rama', 'agha']

Iterate through a list

To loop through a list of elements one at a time, try this. Print names in uppercase using the function upper 
for name in names:
    print(name.upper())
 Or, print only four-letter names. 
for name in names:
    if len(name) == 4:
        print(name)
