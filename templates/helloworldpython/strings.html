Strings

A string in Python is a sequence of characters. It can be a single character, a word or a bunch of text enclosed in quotation marks.

Strings can be declared in single or double quotes. 

print('Python is a great language to learn for beginners.')
print("Python is short and crisp for quickly developing application prototypes.")

Type

There is no character type in Python. There is only str type for both characters and character sequences. 
char = 's'
print(type(char))
Output 
<class 'str'>

Loop

Strings can be iterated like any iterable using the for loop. 
name = 'python'

for letter in name:
    print(letter, end=' ')

print('\n')

#using list comprehension
letters = [letter for letter in name]
print(letters)
Output 
p y t h o n 

['p', 'y', 't', 'h', 'o', 'n']

Length

Use the length function to determine the number of characters in a string. 
word = 'Python'
num_chars = len(word)
print(num_chars)
Output 
 6

Indexing

You can walk through a string using the index operator. String index starts from 0. 
name = "silicon"
first_letter = name[0]
print(first_letter)
last_letter = name[-1]
print(last_letter)
Output 
 
s
n

Slicing

A substring can be made from a string by slicing. A slice of a string expects start and end index within the index operator. 
line = 'Python is a high level, general purpose language.'
language = line[0:6]
print(language)
All letters starting from start index 0 to end index-1.

Output 

Python

Operators for comparison

The opearators == checks for equality and != for inequality. Similarly, the operators < and > for comparison based on lexicographic ordering. Another pair of useful lexicographic comparators are also available - these operators <= and >= compare equality as well. 
s1 = 'python'
s2 = 'Python'

print(s1 == s2)
print(s1 < s2)
print(s1 > s2)
print(s1 <= s2)
print(s1 >= s2)
Output 
False
False
True
False
True
The boolean return True or False helps in decision making during execution of code.

The operators * and + provide useful operations like repetition and concatenation of strings. 

s1 = 'Python'
s2 = 'language'

# repeat P 5 times
p5 = s1[0] * 5
print(p5)

# join s1 and s2 with space in between.
language = s1 + ' ' + s2
print(language)
Output 
PPPPP
Python language

Interpolation

Variables are substituted with their values (interpolated) at runtime using the curly braces { }. 
title = 'Python Basics'
author = 'Nandy OS'
text = f'The book {title} is authored by {author}.'
#note the use of the letter f before the quotation mark. 

print(text)

This is called the f-string. It is useful for string formatting.

Output 
The book Python Basics is authored by Nandy OS.
String formatting is also done using the format method and the % operator, but the f-string is now more common. 
text = "The book {} is authored by {}.".format(title, author)
print(text)
It is, however, not elegant like the f-string.
