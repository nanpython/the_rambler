Dictionary

A dictionary holds one or more key-value pairs in a dict data structure.

The elements in a dictionary are called keys. Each key has a corresponding value. In some languages the dictionary is also known as an associative array. 
capitals = {
    'Telangana' : 'Hyderabad',
    'Tamilnadu' : 'Chennai',
    'Uttar Pradesh' : 'Lucknow'
}

print(type(capitals))
Output

Using the dict() function

Note the difference in the way key-value pairs are written. 
logins = dict(
  admin = "super",
  user = "guest",
  )

print(logins)
print(logins['user'])
Output 
{'admin': 'super', 'user': 'guest'}
guest

Using get() to access a dictionary pair

password = logins.get('admin')
print(password)
# using string formatting
print(f'password for admin is {password}')
Output 
super
When "get()" is called it checks if the given key exists in the dict. If it does not exist then the value of the default argument is returned instead. Helps avoid errors in the code. 
print(logins.get('visitor', 'not found'))
Output 
print(logins.get('visitor', 'not found'))
