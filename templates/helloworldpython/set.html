Set

A set is an unordered collection of unique elements of any type.

Note: Use the embedded python interpreter at the bottom of this page to try the examples.

Examples

Create a set

Create a set of names. 
unique_names = {'modi', 'saboo', 'kumar', 'sam', 'ali'}
 A print of unique_names returns the names in no particular order. 
{'kumar', 'ali', 'modi', 'sam', 'saboo'}
 Create an empty set of entries. 
entries = {}
 As in a list, the elements are separated by commas. Elements can be a string, a number, a boolean value or any object including None.

A set can also be created using the set function. 

a_set = set()
The set function takes only one argument. It can take an iterable as the argument, such as a list, a string or a tuple. 
a_list = ['a', 'b', 'c']
a_set = set(a_list)
print(a_set)
Output 
{'b', 'c', 'a'}

Add an element

Elements can be added or removed from a set. Use the add function to add items to a set.

unique_names.add('hari')
 The element is added to the set. 
{'ali', 'sam', 'saboo', 'modi', 'hari', 'kumar'}

Remove an element

To remove kumar from the set, use remove function. 
unique_names.remove('kumar')
 If you try to remove an element that is not present in the set, then a KeyError is raised. To avoid this error, use the discard function instead. 
unique_names.discard('nandy')

Modify a set

To modify a set, the function update is used. The function takes an iterable such as another set, a list, a string or a tuple. 
print(len(unique_names))
extra_names = {'san', 'lan', 'dan'}
unique_names.update(extra_names)
print(unique_names)
print(len(unique_names))
To remove all elements from a set, use the function clear or simply assign an empty set or another set of elements to replace all.

Iterate through a set

To loop through a set of elements one at a time, try this. 
for item in unique_names:
	print(item)
Or, print only three-letter names. 
for item in unique_names:
    if len(item) <= 3:
        print(item)

Set operations

The set data structure provides functions of the mathematical set, such as
  • union A ∪ B
  • intersection A ∩ B
  • difference A \ B
  • symmetric difference (delta) A Δ B

union of sets

A union of two sets is a collection of elements from both the sets. The operator | is used to perform the union operation. The function union may also be used. 
s1 = {'a', 'b', 'c'}
s2 = {'x', 'y', 'z'}

s3 = s1 | s2
print(s3)
Output 
{'b', 'c', 'a', 'z', 'x', 'y'}

intersection of sets

An intersection of two sets is a set of all those elements that are common to both the sets. The & operator is used to perform the intersection operation. The function intersection may also be used. 
s1 = {'a', 'b', 'c'}
s2 = {'x', 'y', 'b'}

s3 = s1 & s2
print(s3)
Output 
{'b'}

difference in sets

A difference between sets is a set of all elements in the first set that are not present in the second set. The - operator is used to perform the difference operation. The function difference may also be used. 
s1 = {'a', 'b', 'c'}
s2 = {'x', 'y', 'b'}

s3 = s1 - s2
print(s3)
Output 
{'c', 'a'}

symmetric difference in sets

A symmetric difference of sets is a collection of elements of both the sets that are unique. The operator ^ is used to perform the symmetric difference operation. The function symmetric_difference may also be used. 
s1 = {'a', 'b', 'c'}
s2 = {'x', 'y', 'b'}
s3 = s1 ^ s2
print(s3)
Output 
{'a', 'c', 'y', 'x'}
