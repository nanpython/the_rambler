Map

Map is a function to transform every element in a collection without using a loop. It is a functional programming concept.

Note: Use the embedded python interpreter at the bottom of this page to try the examples.

Examples

Convert numbers in a list to strings

Transform a list of names. 
# convert a list of numbers to strings
nums = [12, 34, 45, 56]
str_nums = map(str, nums)
print(str_nums)
print(list(str_nums))
Output 
    <map object at 0x13a550070>
    ['12', '34', '45', '56']

Transforming a range of numbers

Using lambda keyword. 
list(map(lambda x: x*3, range(5)))
Output 
[0, 3, 6, 9, 12]

Transforming strings

Let us say you have a list of names that you want to change to proper nouns (first letter uppercase).

Two approaches:

  • Using list comprehension
  • Using map with lambda function
names = ['anand', 'lakshmi', 'sam', 'raghu', 'sneha']

capitalized_names = [name.capitalize() for name in names]

print(names)
print(capitalized_names)
Output 
['anand', 'lakshmi', 'sam', 'raghu', 'sneha']
['Anand', 'Lakshmi', 'Sam', 'Raghu', 'Sneha']

Using lambda function

The map() function takes two arguments: the transformation function followed by a list.

Here the map() function is used with a lambda function to transform every name in the last. 
mapped_names = list(map(lambda name: name.capitalize(), names))
print(mapped_names)
The lambda keyword is used to create an anonymous function that processes the input and returns a result. In the example above, name is the argument passed on to the anonymous function. it performs the operation, namely capitalize(), placed after the colon.
