Generator

A generator is a function used to create an iterator. The keyword yield is used in a generator to return a value from a function at each iteration.

Note: Use the embedded python interpreter at the bottom of this page to try the examples.

Examples

Create a generator

def sequence(n):
	i=0
	while i < n:
		yield i
		i = i + 1

for k in sequence(5):
	print(k)
Output 
0
1
2
3
4
Note There is no return statement in the generator function. It is replaced by the yield keyword, which returns a value every time the generator function is called.

A generator function can replace a loop construct. when we are dealing with extremely large data that we need to iterate over, a for loop or a while loop would be very inefficient. Instead, a generator function can be very efficient because it will iterator only once when it is called.

Fiboncci series

def fib_seq(n):
    a = 0
    b = 1
    sum = 0
    count = 1
    while(count <= n):
      yield sum
      count += 1
      a = b
      b = sum
      sum = a + b

for f in fib_seq(5):
    print(f, end=' ')
Output 
0 1 1 2 3
